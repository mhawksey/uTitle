INSTALLATION

  1. Register a new application with Twitter
    1. Application Type: Browser
    2. Callback URL: {yourplannedwebaddress}/callback.php
    3. Default Access type: Read & Write
    4. Use Twitter for login: Yes
  2. In config.php
    1. Enter Consumer key and Consumer secret
    2. Enter requested API key from TwapperKeeper
    3. Enter the base url (UTITLE_URL) for your installation
  3. Copy files to server
  4. Chmod 'xml' folder to 777
  5. ... [I think that is everything]
